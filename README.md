# RCJ Robot Soccer (Keisei-RobotSoccer)

We lead open-source robot soccer education from Japan.

## Overview
This repository provides open-source hardware designs, electronics, and software
used in robot soccer (RoboCup Junior level) for educational purposes.

## Contents
- hardware/      : Mechanical design & assembly
- electronics/   : Circuits, wiring, BOM
- software/      : Arduino / micro:bit programs
- docs/          : Manuals (PDF), calibration guides

## Philosophy
- Learn by building
- Open design, explainable code
- Students create, modify, and understand everything
